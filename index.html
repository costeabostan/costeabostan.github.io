SVG Viewer

Useful application to visualise SVG files. The application has the following features:

  • Possibility to convert opened file to PNG;
  • Works offline;
  • Doesn't use web view (uses native code implementation) therefore it consume less battery;
  • Can open SVG files form file explorer applications;
  • Simple to use, doesn't have adds while visualising SVG file, there is only one banner present at settings popup.

SVG Converter

This application allow you to convert SVG file to PNG or JPEG and save it on the device.

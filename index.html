SVG Viewer

Useful application to view SVG files, can show/hide layers of the SVG file.

  • Possibility to convert opened file to PNG;
  • Can open SVG files form file explorer applications;
  • User can choose files from Dropbox and OneDrive using native library implementation;
  • User can visualise recent files that were successfully opened;
  • Possibility to show/hide layers (Layers must have id to appear in the show/hide list).

SVG Converter

This application converts SVG file to PNG or JPEG format and to crop this files.
